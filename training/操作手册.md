# 操作手册

## 第一章 系统登录

### 1.1 访问系统
打开浏览器，输入系统地址：http://your-server.com

### 1.2 登录
1. 输入用户名和密码
2. 点击"登录"按钮
3. 首次登录需修改密码

### 1.3 忘记密码
点击"忘记密码"，通过绑定的手机号或邮箱重置。

---

## 第二章 巡视任务

### 2.1 创建任务
1. 进入"巡视管理" → "任务管理"
2. 点击"新建任务"
3. 填写任务信息：
   - 任务名称
   - 巡视类型
   - 设备选择
   - 计划时间
   - 执行人
4. 点击"保存"

### 2.2 执行任务
1. 进入"我的任务"
2. 选择待执行任务
3. 点击"开始巡视"
4. 按步骤完成巡视项
5. 上传照片/视频
6. 填写巡视结果
7. 点击"完成"

### 2.3 查看任务
1. 进入"巡视管理" → "任务列表"
2. 可按状态、时间、设备等筛选
3. 点击任务查看详情

---

## 第三章 缺陷管理

### 3.1 上报缺陷
1. 在巡视过程中发现缺陷
2. 点击"上报缺陷"
3. 填写缺陷信息：
   - 缺陷类型
   - 缺陷等级
   - 缺陷描述
   - 上传照片
4. 点击"提交"

### 3.2 审核缺陷
1. 进入"缺陷管理" → "待审核"
2. 选择缺陷记录
3. 查看缺陷详情
4. 填写审核意见
5. 选择"通过"或"驳回"

### 3.3 缺陷统计
1. 进入"缺陷管理" → "统计分析"
2. 查看缺陷趋势图
3. 查看缺陷类型分布
4. 导出统计报表

---

## 第四章 工单管理

### 4.1 创建工单
1. 缺陷审核通过后自动/手动创建工单
2. 进入"工单管理" → "新建工单"
3. 填写工单信息：
   - 工单类型
   - 优先级
   - 处理人
   - 要求完成时间
4. 点击"保存"

### 4.2 处理工单
1. 进入"我的工单"
2. 选择待处理工单
3. 点击"开始处理"
4. 填写处理过程和结果
5. 上传处理后照片
6. 点击"完成处理"

### 4.3 验收工单
1. 进入"工单验收"
2. 选择待验收工单
3. 现场/远程验收
4. 填写验收意见
5. 选择"验收通过"或"重新处理"

---

## 第五章 报表中心

### 5.1 巡视报表
1. 进入"报表中心" → "巡视报表"
2. 选择时间范围
3. 查看巡视完成率、及时率
4. 导出 Excel/PDF

### 5.2 缺陷报表
1. 进入"报表中心" → "缺陷报表"
2. 选择时间范围
3. 查看缺陷数量、类型分布
4. 导出 Excel/PDF

### 5.3 工单报表
1. 进入"报表中心" → "工单报表"
2. 选择时间范围
3. 查看工单完成率、及时率
4. 导出 Excel/PDF

---

## 第六章 系统设置

### 6.1 用户管理
1. 进入"系统设置" → "用户管理"
2. 添加/编辑/删除用户
3. 分配角色和权限

### 6.2 设备管理
1. 进入"系统设置" → "设备管理"

3. 维护设备台账

3. 配置通知方式

## 常见问题

**Q: 忘记密码怎么办？**
A: 点击登录页"忘记密码"，通过手机号或邮箱重置。

**Q: 如何导出报表？**
A: 在报表页面点击右上角"导出"按钮，选择 Excel 或 PDF 格式。

**Q: 缺陷等级如何划分？**
A: 分为一般、严重、危急三个等级，根据缺陷对设备运行的影响程度判定。

**Q: 工单超时如何处理？**
A: 系统会自动提醒处理人和上级主管，可在工单详情页申请延期。
