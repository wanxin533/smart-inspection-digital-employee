## 📋 项目简介

智能巡视数字员工系统是一套面向电力、能源、制造等行业的智能化设备巡视管理平台。系统基于 OpenClaw 多 Agent 架构，集成 AI 视觉识别、物联网传感、知识图谱等技术，实现设备巡视的数字化、智能化、自动化。

### 核心价值
- 🎯 **降本增效** - 减少人工巡视工作量 60%+，巡检效率提升 300-500%
- 🔍 **精准识别** - AI 缺陷识别准确率≥96%，漏检率<2%
- 📊 **数据驱动** - 全量巡视数据沉淀，支持趋势分析与预测性维护
- 🔄 **闭环管理** - 缺陷发现 - 派单 - 处理 - 验收全流程数字化

## 🚀 快速开始

### 环境要求
- Docker 20.10+
- MySQL 8.0+
- Redis 7.0+
- Node.js 18+
- Python 3.10+

### 启动服务

```bash
# 克隆项目
git clone https://github.com/wanxin533/smart-inspection-digital-employee.git
cd smart-inspection-digital-employee

# 启动所有服务
docker-compose up -d

# 访问服务
# 前端：http://localhost
# 后端：http://localhost:8080/api
# Swagger: http://localhost:8080/api/swagger-ui.html

# 默认账号：admin / admin123
```
