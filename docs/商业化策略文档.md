# 商业化策略文档

## 1. 产品定位

### 1.1 目标客户
- **电网公司**：国网/南网及省电力公司
- **发电集团**：五大六小及地方能源投资集团
- **电力施工单位**：送变电工程公司/电力建设公司
- **石油化工**：中石油/中石化/中海油
- **轨道交通**：地铁公司/铁路局
- **大型园区**：商业综合体/工业园区/数据中心

### 1.2 核心价值主张
- 降本：人力成本降低 60-72%
- 增效：巡检效率提升 300-500%
- 安全：安全风险降低 80-90%
- 智能：AI 识别准确率≥96%

- 石油化工（15% 收入）
- 轨道交通（10% 收入）
