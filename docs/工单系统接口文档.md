# 工单系统接口文档

## 1. 接口概述

### 1.1 接口列表
| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 创建工单 | POST | /api/work-orders | 创建新工单 |
| 工单列表 | GET | /api/work-orders | 查询工单列表 |
| 工单详情 | GET | /api/work-orders/{id} | 查询工单详情 |
| 分配工单 | POST | /api/work-orders/{id}/assign | 分配工单给处理人 |
| 处理工单 | POST | /api/work-orders/{id}/process | 处理工单 |
| 验收工单 | POST | /api/work-orders/{id}/accept | 验收工单 |
| 关闭工单 | POST | /api/work-orders/{id}/close | 关闭工单 |
| 工单统计 | GET | /api/work-orders/statistics | 工单统计数据 |

---

## 2. 接口详情

### 2.1 创建工单

**请求**：
```json
POST /api/work-orders
{
  "defectId": "DEF20260331001",
  "type": "REPAIR",
  "priority": "HIGH",
  "assigneeId": "USER001",
  "requireTime": "2026-04-02 18:00:00",
  "description": "主变 A 相温度异常，需停电检查"
}
```

**响应**：
```json
{
  "code": 200,
  "data": {
    "id": "WO20260331001",
    "status": "CREATED",
    "createTime": "2026-03-31 14:00:00"
  }
}
```

### 2.2 工单列表

**请求**：
```
GET /api/work-orders?status=PROCESSING&assigneeId=USER001&page=1&size=20
```

**响应**：
```json
{
  "code": 200,
  "data": {
    "total": 156,
    "list": [
      {
        "id": "WO20260331001",
        "type": "REPAIR",
        "priority": "HIGH",
        "status": "PROCESSING",
        "assigneeName": "张三",
        "createTime": "2026-03-31 14:00:00"
      }
    ]
  }
}
```

### 2.3 分配工单

**请求**：
```json
POST /api/work-orders/WO20260331001/assign
{
  "assigneeId": "USER002",
  "comment": "请优先处理"
}
```

**响应**：
```json
{
  "code": 200,
  "data": {
    "success": true,
    "assignTime": "2026-03-31 14:30:00"
  }
}
```

### 2.4 处理工单

**请求**：
```json
POST /api/work-orders/WO20260331001/process
{
  "processResult": "已更换套管，温度恢复正常",
  "photos": ["photo1.jpg", "photo2.jpg"],
  "consumeTime": 120
}
```

**响应**：
```json
{
  "code": 200,
  "data": {
    "success": true,
    "processTime": "2026-03-31 16:00:00"
  }
}
```

### 2.5 验收工单

**请求**：
```json
POST /api/work-orders/WO20260331001/accept
{
  "acceptResult": "PASS",
  "comment": "处理合格，温度正常",
  "photos": ["accept1.jpg"]
}
```

**响应**：
```json
{
  "code": 200,
  "data": {
    "success": true,
    "acceptTime": "2026-03-31 17:00:00"
  }
}
```

### 2.6 工单统计

**请求**：
```
GET /api/work-orders/statistics?startTime=2026-03-01&endTime=2026-03-31
```

**响应**：
```json
{
  "code": 200,
  "data": {
    "total": 1250,
    "byStatus": {
      "CREATED": 50,
      "PROCESSING": 80,
      "ACCEPTED": 1000,
      "CLOSED": 120
    },
    "byPriority": {
      "HIGH": 150,
      "MEDIUM": 600,
      "LOW": 500
    },
    "avgProcessTime": 4.5,
    "onTimeRate": 98.5
  }
}
```

## 3. 错误码

| 错误码 | 说明 |
|--------|------|
| 200 | 成功 |
| 400 | 请求参数错误 |
| 401 | 未授权 |
| 403 | 无权限 |
| 404 | 工单不存在 |
| 500 | 服务器内部错误 |
