# 智能巡视数字员工 - 需求规格说明书 (PRD) v1.0

## 1. 产品概述

### 1.1 产品定位
智能巡视数字员工系统是一套面向电力、能源、制造等行业的智能化设备巡视管理平台。

### 1.2 目标用户
- **设备专责**：制定巡视标准、审核计划、分析缺陷趋势
- **设备主人**：分配任务、跟踪进度、初审缺陷、派发工单
- **现场运维**：执行巡视任务、上报缺陷、处理工单

### 1.3 核心价值
- 降本增效：减少人工巡视工作量 60%+
- 精准识别：AI 缺陷识别准确率≥96%
- 数据驱动：全量数据沉淀，支持预测性维护
- 闭环管理：缺陷 - 派单 - 处理 - 验收全流程数字化

## 2. 功能需求（15 项 P0 功能）

### F001 用户登录认证
- **功能描述**：支持账号密码、短信验证码登录，会话管理
- **用户角色**：全部
- **验收标准**：登录成功率≥99%，会话超时 30 分钟

### F002 巡视任务列表
- **功能描述**：展示待执行、进行中、已完成任务，支持筛选
- **用户角色**：现场运维
- **验收标准**：列表加载时间<1 秒，筛选响应<500ms

### F003 巡视任务执行
- **功能描述**：按步骤引导完成巡视，支持拍照、录入数据
- **用户角色**：现场运维
- **验收标准**：离线可用，数据自动同步

### F004 AI 图像识别
- **功能描述**：自动识别设备缺陷（表计读数、外观异常等）
- **用户角色**：全部
- **验收标准**：识别准确率≥96%，响应时间<3 秒

### F005 缺陷上报
- **功能描述**：手动或 AI 触发缺陷上报，填写缺陷详情
- **用户角色**：现场运维
- **验收标准**：上报流程<1 分钟完成

### F006 缺陷审核
- **功能描述**：审核缺陷报告，确认缺陷等级和处理方式
- **用户角色**：设备主人
- **验收标准**：审核时效<2 小时

### F007 工单创建
- **功能描述**：基于缺陷自动生成工单，指派处理人员
- **用户角色**：设备主人
- **验收标准**：工单自动生成率 100%

### F008 工单处理
- **功能描述**：接收工单，记录处理过程和结果
- **用户角色**：现场运维
- **验收标准**：处理记录完整率 100%

### F009 工单验收
- **功能描述**：验收工单处理结果，关闭工单
- **用户角色**：设备主人
- **验收标准**：验收时效<4 小时

### F010 设备台账查看
- **功能描述**：查看设备基本信息、历史巡视记录
- **用户角色**：全部
- **验收标准**：查询响应<1 秒

### F011 巡视计划管理
- **功能描述**：创建、编辑、发布周期性巡视计划
- **用户角色**：设备专责
- **验收标准**：计划准确率 100%

### F012 任务自动派发
- **功能描述**：根据计划自动生成并派发巡视任务
- **用户角色**：系统自动
- **验收标准**：派发及时率 100%

### F013 消息通知
- **功能描述**：任务提醒、缺陷告警、工单通知（APP 推送）
- **用户角色**：全部
- **验收标准**：通知到达率≥99%

### F014 个人工作台
- **功能描述**：展示待办任务、通知、快捷入口
- **用户角色**：全部
- **验收标准**：个性化配置保存

### F015 基础报表
- **功能描述**：巡视完成率、缺陷统计等基础数据报表
- **用户角色**：设备主人
- **验收标准**：报表生成时间<5 秒

## 3. 非功能需求

### 3.1 性能要求
- 响应时间：<500ms（95% 请求）
- 并发用户：≥1000
- 吞吐量：≥100 QPS

### 3.2 安全要求
- 认证：JWT Token + 刷新机制
- 授权：RBAC 权限模型
- 加密：TLS 1.3 + AES-256

### 3.3 可用性要求
- SLA：99.9%
- 灾备：RTO<4 小时，RPO<15 分钟
- 监控：全链路监控告警

### 3.4 兼容性要求
- 浏览器：Chrome/Firefox/Safari/Edge 最新 2 版本
- 移动端：iOS 13+ / Android 10+
- 分辨率：1920×1080 / 1366×768 / 移动端自适应

## 4. 数据需求

### 4.1 设备台账（19 字段）
设备 ID、设备名称、设备类型、电压等级、生产厂家、投运日期、安装位置、设备状态、上次检修日期、下次检修日期、健康评分、缺陷数量、巡视次数、负责人、备注等

### 4.2 巡视记录（15 字段）
记录 ID、任务 ID、设备 ID、巡视人员、巡视时间、巡视结果、缺陷数量、照片数量、GPS 定位、环境数据、备注、审核状态、审核人员、审核时间等

### 4.3 缺陷数据（20 字段）
缺陷 ID、设备 ID、缺陷类型、缺陷等级、缺陷描述、发现人员、发现时间、照片/视频、位置信息、处理建议、处理人员、处理时间、验收人员、验收时间、状态等

### 4.4 工单数据（23 字段）
工单 ID、缺陷 ID、工单类型、工单状态、优先级、指派人员、创建时间、要求完成时间、实际完成时间、处理过程、处理结果、验收意见、附件、延期原因等
