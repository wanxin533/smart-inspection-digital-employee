# 模型训练报告

## 1. 训练概述

### 1.1 训练目标
- YOLOv10：8 类缺陷检测，mAP50 ≥ 0.85

- LLaVA-1.6：截图分析，准确率≥90%

### 1.2 训练环境
- GPU：NVIDIA A100 × 4

- CUDA：11.8

## 3. PatchCore 异常检测模型
