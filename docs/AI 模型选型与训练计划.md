# AI 模型选型与训练计划

## 1. AI 模型选型

### 1.1 缺陷检测模型
**选型**: YOLOv10
- 最新架构，无 NMS
- 推理速度：85 FPS
- 预期指标：mAP50 ≥ 0.85

### 1.2 异常检测模型
**选型**: PatchCore
- 无监督异常检测
- 工业场景验证充分
- 预期指标：AUROC ≥ 0.92

### 1.3 VLM 模型
**选型**: LLaVA-1.6-7B
- 开源可商用
- 中文支持好
- 支持 8bit 量化

### 2.1 数据收集
- 历史巡检图片：10,000+ 张

## 3. 模型训练计划

### 3.1 训练环境
- GPU: NVIDIA A100 × 4
- 显存：40GB × 4
- 框架：PyTorch 2.0

### 3.2 训练周期
- YOLOv10: 4 周
- PatchCore: 3 周
- LLaVA 微调：2 周

### 3.3 评估指标
- 准确率、召回率、F1
- mAP50、mAP50-95
- AUROC
