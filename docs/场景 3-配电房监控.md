# 场景 3-配电房环境监控实施方案

## 1. 场景需求分析

### 1.1 业务痛点
- 环境异常发现滞后（水浸/火灾/气体泄漏）
- 人工巡检频次低（1 次/天，无法实时）
- 多系统孤立（温湿度/安防/消防独立）
- 联动控制缺失（空调/风机/水泵手动）

### 1.2 监测指标
- 环境参数（温度/湿度/水浸）
- 安全参数（烟雾/气体浓度/门禁）
- 设备参数（开关状态/电流电压）
- 视频参数（实时监控/入侵检测）

---

## 2. 技术方案

### 2.1 系统架构
```
IoT 传感器 → 边缘网关 → 4G/光纤 → 监控平台 → 联动控制
```

### 2.2 设备配置
- 温湿度传感器：8 个（每面墙 2 个）
- 水浸传感器：4 个（电缆沟/门口）
- 烟雾传感器：4 个（天花板）
- SF6 气体传感器：2 个（开关柜）
- 边缘网关：1 台（边缘计算 + 联动控制）
- 摄像头：6 个（全景 + 细节）

### 2.3 联动策略
- 温度>40°C → 启动空调/风机
- 湿度>80% → 启动除湿机
- 水浸告警 → 启动水泵 + 关闭进水阀
- 烟雾告警 → 启动消防 + 切断电源
- 气体泄漏 → 启动风机 + 声光告警
